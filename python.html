🧠 Project #1 – GeoLab Pro

GeoLab Pro is a professional-grade, bilingual (Bengali + English) toolkit built for Geology students, researchers, and educators. It streamlines key geological calculations such as True Dip, Porosity, Stratigraphic Thickness, Slope Gradient, and Grain Size conversions. All tools are available in one interactive online interface.

📘 Example Use Case

Tool: True Dip Calculator

Input: Apparent Dip = 30°, Angle Between Directions = 60°

Output: True Dip = 33.69°

Formula: True Dip = tan⁻¹(tan(Apparent Dip) / sin(Angle))

This tool is vital for visualizing 3D geological structures. It enables structural geologists and field teams to interpret real dip angles quickly and accurately.
